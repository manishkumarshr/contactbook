-- SUMMARY --
A module that provides and interface to store your contact as phone book and manage it.

-- REQUIREMENTS --
None.

-- Installation --
To install, copy the contactbook directory and all its contents to your modules directory.
To enable this module, go to Administer > Modules, and enable "Contact Book".


-- CONFIGURATION --
* Go To admin/config/contactbook/
* Add the default Country Calling Code that will be used as prefix for all phone numbers. You can edit that while creating a contact.
